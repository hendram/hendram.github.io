Freelancer Management System

Intelligent freelancer discovery, ranking, and task-matching platform.

Inspiration

This project was inspired by platforms like Upwork and Freelancer, as well as real-world observations from developer peers. Many developers share similar years of experience but specialize in different domains. One may excel in building product catalogs while another specializes in payment gateway integration.

The idea was simple: what if experts collaborated only within their strongest domains? This benefits everyone:

  • Managers get the best talent for each task.
  • Freelancers focus only on their strongest expertise.
  • Companies receive higher-quality software built by specialists.

What It Does

The system builds a structured freelancer database where each freelancer can refer others for specific user stories. Managers can assign reputation scores after tasks, and search freelancers using skill filters, rankings, and performance history.

Managers can also send RFPs, receive proposals, negotiate fees, and select the most suitable freelancer for a task.

How It Was Built

  • Developed using Atlassian Forge (Run on Atlassian compliant)
  • GlobalPage integrated with Jira IssuePanel module
  • Custom UI frontend
  • TiDB used as storage
  • Jira API used to retrieve user story data

Candidate Scoring Logic

Experience Score

1 year experience = 10 points

Experience Score = Years × 10

Reputation Score

  • Assigned manually by manager
  • No multiplier
  • No decay
  • Represents completed task performance

Total Capability Score

Total = Experience Score + Reputation Score

Task Requirement

1 Story Point = 10 Required Points

Required Score = Story Points × 10

Eligibility Rule

Freelancer Eligible if → Total ≥ Required
Only eligible freelancers are ranked. Top 5 highest scores are shown.
